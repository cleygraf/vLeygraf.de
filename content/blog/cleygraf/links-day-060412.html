  • Verschlüsselung von Clouddaten

Grundsätzlich bereiten mir die unverschlüsselten Daten in meinem Dropbox-Folder Unbehagen. Es gibt aber einige Lösungsansätze: BoxCryptor bzw. EncFS. Nachteil dieser Lösung ist, dass dafür üblicherweise ein Kernelmodul (LINUX und Mac OS X) notwendig ist.

Gerade bin ich auf SecretSync gestossen. Dabei scheint es sich um eine lokalen Synchronisationsmechanismus zu handeln, der auf Kernelmodule u. ä. verzichtet. Das Ganze ist mit JAVA umgesetzt, sodass es auch unter LINUX nutzbar ist. Leider aber nicht Opensource.
