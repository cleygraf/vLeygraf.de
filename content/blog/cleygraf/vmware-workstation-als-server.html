--- kind: article title: VMware Workstation als Server created_at: 2010-08-29 author: Christoph Leygraf ---

Letzte Woche habe ich über eine Nachfolgelösung für VMware Server nachgedacht. Wenn man jedoch nach "VMware Workstation headless" googelt, lernt man, wie man auch die Workstation-Variante als Server missbrauchen kann.
Ich benutzte eine aktuelle Ubuntu 64bit Installation mit VMware Workstation 7.1. Für die VM habe ich unter "Options" "Remote Display" aktiviert. Auch wenn man sich lokal nicht angemeldet hat, kann man jetzt mit folgendem Kommando über ein SSH-Login die VM starten:

vmrun -T ws start ~/vmware/Meine_VM/VM.vmx nogui

Wichtig ist der Parameter "nogui". Anschließend kann man einen VNC-Viewer benutzen (Verbindung zur IP/Hostnamen des Ubuntu-Systems mit dem in der VM-Konfiguration angegebenen Port) um der VM beim Starten zusehen. Natürlich ist auch ein Login möglich.
Klasse!

Nachtrag - Achtung:
Loggt man sich danach graphisch (also z. B. lokal mit GDM oder remote mit NoMachine) ein und startet VMware Workstation, so wird die laufenden VM übernommen. Beim Ausloggen wird die VM dann ausgeschaltet und läuft nicht weiter!
